# asdf-netwatch plugin for the  [asdf version manager](https://asdf-vm.com/)

Real-time network diagnostics in your terminal. One command, zero config, instant visibility.  

https://github.com/matthart1983/netwatch

## Install plugin
```bash
asdf plugin add netwatch https://github.com/a1xs/asdf_netwatch
```
#  Install app

## Install latest version
```bash
asdf install netwatch latest
```

## Install specific version
```bash
asdf install netwatch 0.12.5
```
